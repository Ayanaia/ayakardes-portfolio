Web Development

Project ArtNation

I built a niche community website using the Laravel Framework, aimed at uniting users around shared passions and interests. The platform supports user-generated content, enabling members to contribute articles, forum posts, and personalized features that enhance engagement. Community interaction is facilitated through comments, likes, and direct messaging, allowing users to connect meaningfully. The website includes robust functionalities like user authentication, a well-structured database with migrations, search capabilities, and an administrative backend powered by Laravel Filament for easy management.

Travel Planner

In this project, I created a Single Page Application (SPA) for a Travel Planner linked to a custom-built API that stores data in MongoDB. The application allows users to register, log in, and manage their travel trips, including creating, updating, and viewing trip details. Users can track activities, add notes, and monitor expenses for each trip, all while enjoying a visually appealing and responsive user interface. Additional features include location mapping, and notifications for upcoming trips, enriching the travel planning experience.

Handmade Bag Webshop

I developed a webshop platform that allows users to create accounts, log in, and manage their shopping experience seamlessly. Users can search and filter products based on categories and price, add items to their favorites and shopping carts, and proceed to checkout. The platform features an intuitive onboarding process, allowing visitors to register, view their profiles, and contact sellers for inquiries. As a seller, users can manage their products, respond to messages, and update their profiles through a user-friendly dashboard.

Waves: Music For Everyone

I built a basic HTML and CSS skeleton for a music streaming platform, focusing on a clean and responsive design. This structure allows users to navigate through music catalogs and create playlists, providing a solid foundation for future functionality and enhancements.
